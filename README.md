# Simple Calculator

A beginner-friendly command-line calculator built using Python.  
This project helped me understand basic programming concepts like user input, functions, and condition handling.

---

##  Features

- Addition  
- Subtraction  
- Multiplication  
- Division  
- Handles division by zero  
- Simple and clean CLI interface  

---

## ▶ How to Run

1. Open terminal  
2. Navigate to the project folder  
3. Run the following command:

python main.py
